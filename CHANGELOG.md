## v[1.0] - Initial Release(Updated)

### Added

* Wallpaper slideshow engine
* Random and sequential wallpaper rotation
* Wallpaper preview gallery in the GUI
* Folder selection for wallpapers
* Configurable interval for wallpaper changes
* Configuration file support (~/.config/wallpy/config.json)

### GUI

* Lightweight interface built using Tkinter
* Wallpaper thumbnail previews
* Start/Stop wallpaper engine controls

### Packaging

* Standalone binary build using PyInstaller
* Portable AppImage distribution

* Open-source GitHub repository
* Project documentation (README)
